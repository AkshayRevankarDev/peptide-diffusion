# Raw MS/MS Data Directory

This directory is intended to store the raw spectral files and Database Search outputs.
Since spectral files like `.mzML` are very large (several GBs), they **are not tracked by version control**.

To reproduce this project, please obtain the raw data files (`Ecoli_EV_1.mzML` and `Database search output_Ecoli_EV_1.xlsx`) and place them in this directory or adjust the dataset paths.
