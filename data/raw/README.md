# Raw MS/MS Data Directory

This directory stores raw spectral files and database search outputs.
These files are **not tracked by git** (too large for GitHub — mzML files are 500 MB–1 GB each).

## Download the data

All raw data files are available on Google Drive:
**https://drive.google.com/drive/folders/10_67wgMzvbi0jUoLoW7VTXNL3q63BWtL**

Download and place the following files in this directory:

| File | Description |
|------|-------------|
| `Ecoli_EV_1.mzML` | E. coli EV Orbitrap spectra (replicate 1) |
| `Ecoli_EV_2.mzML` | E. coli EV Orbitrap spectra (replicate 2) |
| `Database search output_Ecoli_EV_1.xlsx` | Ground-truth PSM labels (replicate 1) |
| `Database search output_Ecoli_EV_2.xlsx` | Ground-truth PSM labels (replicate 2) |
| `wastewater_Sample1_1.mzML` | Wastewater sample 1, replicate 1 |
| `wastewater_Sample1_2.mzML` | Wastewater sample 1, replicate 2 |
| `wastewater_Sample2_1.mzML` | Wastewater sample 2, replicate 1 |
| `wastewater_Sample2_2.mzML` | Wastewater sample 2, replicate 2 |
